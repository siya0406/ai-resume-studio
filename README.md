# 🚀 AI Resume Studio

An AI-powered resume enhancement and analysis web application built using **Next.js, TypeScript, and Gemini API**.  
It allows users to upload resumes, improve them using AI, and download optimized versions in multiple formats.

---

## ✨ Features

- 📄 Upload Resume (PDF, DOC, DOCX, TXT)
- 🤖 AI-powered Resume Enhancement using Gemini API
- 📊 ATS-friendly Resume Optimization
- 📥 Download Resume as TXT
- 📥 Download Resume as PDF
- 📋 Copy enhanced resume output
- ⚡ Fast processing with server-side API routes
- 📱 Fully responsive UI (mobile + desktop)
- 🎨 Modern UI with Tailwind CSS

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router), React, TypeScript  
- **Styling:** Tailwind CSS  
- **AI Integration:** Google Gemini API  
- **PDF Generation:** jsPDF  
- **Backend:** Next.js API Routes  

---

## ⚙️ How It Works

1. User uploads resume file  
2. File is converted into text  
3. Text is sent to Gemini AI API  
4. AI returns improved ATS-friendly resume  
5. User can copy or download result as TXT/PDF
